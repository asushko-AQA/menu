# Аудитория и ограничения

Критерии используются при разработке меню и блюд.

## Аудитория

| Параметр | Значение |
|----------|----------|
| **Семья** | 5 человек |
| **Взрослые** | 2 |
| **Дети** | 3 (10 лет, 4.5 года, 2.5 года) |
| **Особенности** | У одного ребёнка частичная лактозная непереносимость |

## Ограничения по блюдам

- **Время приготовления:**
  - **Завтрак:** не более 40 минут.
  - **Ужин:** до 60 минут; крайне редко (лучше на выходных) — до 90 минут.
- **Бюджет:** без супердорогих продуктов.
- **Молочные продукты:** только безлактозные (молоко, творог, сыр) либо альтернативы без лактозы.

## Цель по завтракам

- Минимум **15–20 вариантов** завтраков.
- При чередовании — **не повторяться минимум 2 недели**.

## Расширение

При добавлении новых приёмов пищи (обед, ужин и т.д.) можно завести отдельный файл критериев или дополнять этот — по согласованию в [README](../../README.md).
