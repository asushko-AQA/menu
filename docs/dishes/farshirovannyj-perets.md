# Фаршированный перец

Сладкий перец наполняют фаршем с рисом, тушат в томатном соусе. Классическое второе. Рис отваривают до полуготовности или используют сырой — тогда время тушения дольше.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Перец сладкий крупный | 6–8 шт. |
| Фарш (говядина или свино-говяжий) | 550–650 г |
| Рис круглый | 130–150 г (сухой) |
| Лук репчатый | 1 шт. (100 г) |
| Томатная паста или томаты в соку | 2–3 ст. л. / 300–400 г |
| Вода | 300–450 мл |
| Масло растительное | 2 ст. л. |
| Соль, перец | по вкусу |
| Сахар | 0,5 ч. л. |
| Сметана БЛ при подаче | по желанию |
| Зелень | по вкусу |

## Специи и приправы

- Соль, перец чёрный. Лавровый лист — 1–2 шт. В фарш: соль, перец, можно чеснок. В соус: сахар для баланса кислоты.

## Приготовление

1. Рис отварить до полуготовности (10–12 мин), откинуть. Лук мелко нарезать, смешать с фаршем, рисом, солью и перцем. У перцев срезать «крышечку», вынуть семена.
2. Нафаршировать перцы мясной начинкой. В кастрюлю с толстым дном налить масло, уложить перцы вертикально. Развести томатную пасту водой (или использовать томаты в соку), залить перцы до половины. Добавить соль, сахар, лавровый лист.
3. Довести до кипения, накрыть крышкой и тушить на слабом огне 40–50 мин до мягкости перца и готовности риса. Подавать с соусом и сметаной БЛ по желанию.

## Подготовка заранее

- **С вечера:** отварить рис, нарезать лук, смешать фарш с рисом и луком; вымыть перцы и вынуть семена. Утром только нафаршировать и тушить. Либо нафаршировать перцы с вечера и убрать в холодильник — вечером поставить тушиться.

## Состав по компонентам

- **Начинка:** фарш, рис, лук.
- **Основа:** перец сладкий, томатный соус.
- **Подача:** перец в соусе. Сметана БЛ, зелень, хлеб.

## Порции и время

- На 5 человек. Около 50–60 мин (с учётом тушения). Заготовка с вечера сокращает время.

## Аллергены и особенности

- Без молочного в блюде. Без глютена. Сметана при подаче — только БЛ.

## Подача

- Горячие фаршированные перцы с соусом. Детям — по 1–2 перца на порцию.
