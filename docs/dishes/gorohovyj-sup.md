# Гороховый суп

Суп из колотого гороха с мясом или копчёностями, картофелем и зажаркой. Сытный, хорошо хранится 2–3 дня. Копчёности — по желанию (можно без них).

## Ингредиенты (на 15 порций — 5 чел. × 3 дня)

| Продукт | Количество |
|---------|------------|
| Горох колотый (жёлтый или зелёный) | 300–400 г |
| Говядина или свинина (лопатка, грудинка) | 400–500 г |
| Копчёности (рёбрышки, грудинка) — по желанию | 150–200 г |
| Картофель | 400–500 г |
| Морковь | 200–250 г |
| Лук репчатый | 200–250 г |
| Томатная паста | 1–2 ст. л. |
| Масло растительное | 2 ст. л. |
| Вода | 3–3,5 л |
| Соль | по вкусу |
| Зелень (укроп, петрушка) | по вкусу |
| Сухарики или хлеб при подаче | по вкусу |

## Специи и приправы

- Соль, перец чёрный горошком — 2–3 шт. Лавровый лист — 1–2 шт. Чеснок — 1–2 зубчика (в конце). Зелень — при подаче.

## Приготовление

1. Горох промыть, залить холодной водой на 2–4 ч или варить без замачивания дольше. Мясо (и копчёности при желании) залить холодной водой, довести до кипения, снять пену. Варить 1–1,5 ч. Горох можно варить вместе с мясом: засыпать после закипания и варить до мягкости гороха (40–60 мин в зависимости от сорта).
2. Мясо вынуть, нарезать кусочками. Картофель нарезать кубиками, добавить в суп. Варить 15–20 мин до мягкости картофеля.
3. Зажарка: лук и морковь нарезать, обжарить на масле 5 мин, добавить томатную пасту, потушить 2 мин. Выложить в суп, добавить лавровый лист, перец, соль. Варить 5–10 мин. Вернуть мясо. Чеснок — раздавить и добавить в конце.
4. Хранить в холодильнике 2–3 дня. Разогревать порциями. При разогреве можно подлить немного воды — горох загущает. Подавать с зеленью и сухариками.

## Подготовка заранее

- **В день варки:** с вечера замочить горох в воде. Нарезать морковь и лук — хранить в холодильнике. Утром сварить бульон с горохом и собрать суп.

## Состав по компонентам

- **Основа:** горох колотый, мясо, при желании копчёности.
- **Овощи:** картофель, морковь, лук.
- **Зажарка:** морковь, лук, томатная паста.
- **Подача:** зелень, сухарики, хлеб.

## Порции и время

- На 15 порций (5 человек × 3 дня). Время: ~50–60 мин (с замачиванием гороха быстрее). Хранение в холодильнике 2–3 дня.

## Аллергены и особенности

- Без глютена. Без молочного. Копчёности — по желанию (детям можно меньше или без них).

## Подача

- Горячий суп. Зелень и сухарики в тарелку. Хлеб.
