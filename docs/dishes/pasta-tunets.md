# Паста с тунцом

Спагетти или пенне с консервированным тунцом, томатами и чесноком. Быстро, сытно, без сливок.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Паста (спагетти, пенне, фузилли) | 350–400 г |
| Тунец в собственном соку (консервы) | 2 банки (около 400 г) |
| Томаты консервированные (нарезка или в соку) | 300–400 г |
| Лук репчатый | 1 шт. (80–100 г) |
| Чеснок | 2–3 зубчика |
| Оливковое или растительное масло | 2–3 ст. л. |
| Соль, перец | по вкусу |
| Сахар | 0,5 ч. л. (смягчает кислоту) |
| Базилик сушёный или свежий | 0,5 ч. л. / несколько листиков |
| Петрушка или укроп (при подаче) | по вкусу |
| Сыр БЛ тёртый (при подаче, по желанию) | 50–80 г |

## Специи и приправы

- Соль, перец чёрный. Базилик, чеснок. Орегано — по желанию (щепотка). Красный перец хлопьями — по желанию (детям не добавлять). Сахар в соус — для баланса кислоты.

## Приготовление

1. Пасту отварить по инструкции на упаковке до аль денте. Лук нарезать кубиками, чеснок измельчить. Тунец размять вилкой, слить лишний сок (можно оставить немного для соуса).
2. На сковороде разогреть масло, обжарить лук до мягкости. Добавить чеснок, через 30 сек — томаты (раздавить или нарезать), соль, перец, сахар и базилик. Тушить 5–7 мин до загустения. Добавить тунца, прогреть 1–2 мин.
3. Смешать пасту с соусом в сковороде или выложить пасту в тарелки и полить соусом. Посыпать зеленью и сыром БЛ при желании. Подавать сразу.

## Состав по компонентам

- **Основа:** паста, тунец консервированный, томаты, лук, чеснок.
- **Подача:** паста с соусом. Сыр БЛ по желанию.

## Порции и время

- На 5 человек. Около 20–25 мин. Соус можно приготовить пока варится паста.

## Аллергены и особенности

- Глютен (паста). Рыба (тунец). Без молочного в соусе. Сыр при подаче — только БЛ.

## Подача

- Горячая паста с соусом. Классика — сыр сверху.
