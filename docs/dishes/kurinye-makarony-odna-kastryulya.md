# Куриные макароны «в одной кастрюле»

Курицу обжаривают, засыпают сухими макаронами, заливают бульоном или водой и тушат до готовности. Всё в одной посуде.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Куриное филе или бедро без кости | 500–600 г |
| Макароны (рожки, перья) | 350–400 г |
| Лук репчатый | 1–2 шт. (150–200 г) |
| Морковь | 1 шт. (80–100 г) |
| Томатная паста | 2 ст. л. |
| Вода или бульон | 800–1000 мл |
| Масло растительное | 2 ст. л. |
| Соль, перец | по вкусу |
| Сыр БЛ тёртый (при подаче, по желанию) | 80–100 г |

## Специи и приправы

- Соль, перец чёрный. Паприка сладкая — 0,5 ч. л. Чеснок — 1–2 зубчика. Лавровый лист — 1 шт. Сушёная зелень (укроп, петрушка) — по вкусу.

## Приготовление

1. Курицу нарезать кубиками. Лук и морковь нарезать кубиками. В кастрюле с толстым дном разогреть масло, обжарить курицу 5–6 мин, добавить лук и морковь, обжарить ещё 3–4 мин.
2. Добавить томатную пасту, перемешать. Всыпать макароны, залить водой или бульоном так, чтобы жидкость покрывала макароны на 1–1,5 см. Добавить соль, перец и специи. Довести до кипения, убавить огонь, накрыть крышкой и варить 12–15 мин, помешивая, до готовности макарон и выпаривания лишней жидкости.
3. Подавать горячим. По желанию посыпать тёртым сыром БЛ. Один горшок — минимум мытья.

## Состав по компонентам

- **Основа:** куриное мясо (кубики), макароны, бульон/вода, лук, морковь.
- **Подача:** макароны с курицей в тарелках.

## Порции и время

- На 5 человек. Около 30–35 мин. Один горшок — минимум мытья.

## Аллергены и особенности

- Глютен (макароны). Без молочного в блюде. Сыр при подаче — только БЛ.

## Подача

- Горячее, сразу из кастрюли. Можно посыпать сыром БЛ при подаче.
