# Банановые оладьи (без сахара)

Оладьи из банана, яйца и небольшого количества муки. Сладкие за счёт банана, без добавления сахара.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Бананы спелые | 3–4 шт. (350–450 г) |
| Яйца | 2–3 шт. |
| Мука пшеничная | 80–120 г |
| Соль | щепотка |
| Масло для жарки | 25–35 г |
| Сметана БЛ, мёд или ягоды для подачи | по вкусу |

## Специи и приправы

- Соль. Корица или ванилин — по желанию (щепотка в тесто). Подача: мёд, ягоды, сметана БЛ.

## Приготовление

1. Бананы размять вилкой в пюре. Добавить яйца, соль, перемешать. Всыпать муку и замесить тесто консистенции густой сметаны (бананы бывают разной влажности — муку подбирать по ситуации).
2. Разогреть сковороду с маслом. Выкладывать тесто столовой ложкой, формируя небольшие оладьи. Жарить на среднем огне 2–3 мин с каждой стороны до золотистой корочки.
3. Подавать тёплыми со сметаной БЛ, мёдом или ягодами.

## Состав по компонентам

- **Основа:** банан, яйцо, мука.
- **Подача:** оладьи со сметаной БЛ, мёдом или ягодами.

## Порции и время

- На 5 человек. Около 20 мин.

## Аллергены и особенности

- Яйцо. Глютен (мука). Без молочного в тесте. Банан — редко аллерген.

## Подача

- Тёплые оладьи. Подходят детям любого возраста.
