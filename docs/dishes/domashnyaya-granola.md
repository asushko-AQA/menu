# Домашняя гранола

Овсянку с мёдом и семечками запекают заранее; утром достаточно залить БЛ молоком. Хрустящий полезный завтрак.

## Ингредиенты (на 5 человек — запас на несколько дней)

| Продукт | Количество |
|---------|------------|
| Овсяные хлопья (долгой варки) | 300–400 г |
| Мёд | 80–100 г |
| Растительное масло | 40–50 мл |
| Семечки подсолнечника | 50–70 г |
| Семечки тыквенные | 30–50 г |
| Соль | щепотка |
| Изюм или сухофрукты (после запекания) | 50–80 г |
| Молоко БЛ для подачи | по порциям |

## Специи и приправы

- Соль. Корица молотая — 0,5–1 ч. л. Ванилин — на кончике ножа. По желанию: мускатный орех, имбирь молотый — щепотка.

## Приготовление

1. Духовку разогреть до 150–160 °C. Смешать овсянку с семечками, солью и специями. Отдельно смешать мёд с маслом (можно слегка подогреть для текучести), залить сухую смесь и хорошо перемешать.
2. Выложить массу на противень с пергаментом ровным слоем. Запекать 25–35 мин, каждые 10 мин перемешивать, чтобы подрумянилась равномерно. Не пережечь — гранола темнеет и горчит.
3. Достать, остудить, при желании добавить изюм или нарезанные сухофрукты. Хранить в банке. Утром порцию залить БЛ молоком (или йогуртом) и подавать.

## Подготовка заранее

- **Впрок (раз в 1–2 недели):** запечь гранолу по рецепту, остудить, хранить в банке. Утром только отмерить порцию и залить БЛ молоком или йогуртом — готово за минуту.

## Состав по компонентам

- **Основа:** овсяные хлопья, мёд, семечки, масло.
- **Подача:** гранола + БЛ молоко (или йогурт). По желанию — сухофрукты.

## Порции и время

- Запекание: 30–40 мин один раз. Утром: 0 мин, залить молоком. На 5 человек — запас на неделю.

## Аллергены и особенности

- Глютен (овёс). Мёд. Семечки. Молоко — только БЛ. Орехи — при добавлении. Мёд — детям старше года.

## Подача

- Холодная гранола с холодным или слегка подогретым БЛ молоком. Хранить в банке.
