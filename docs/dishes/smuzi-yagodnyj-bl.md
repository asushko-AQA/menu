# Смузи ягодный на БЛ йогурте

Бананы, замороженные ягоды и БЛ йогурт (или молоко) взбивают в блендере. Густой холодный смузи на полдник. Готовится за 5 минут. Можно добавить овсяные хлопья для сытности.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Банан | 2–3 шт. (250–300 г) |
| Ягоды замороженные (малина, черника, клубника) | 200–300 г |
| Йогурт БЛ или молоко БЛ | 300–400 мл |
| Мёд (по желанию) | 1–2 ст. л. |
| Овсяные хлопья (по желанию, для сытности) | 2–3 ст. л. |

## Специи и приправы

- Не требуются. Мёд — по вкусу. Ванилин — на кончике ножа (по желанию).

## Приготовление

1. Банан нарезать кусочками. Ягоды разморозить 5–10 мин или использовать сразу из морозилки (смузи будет холоднее).
2. В блендер положить банан, ягоды, йогурт (или молоко), при желании мёд и овсяные хлопья. Взбить до однородности. Если слишком густо — добавить молока или воды.
3. Разлить по стаканам. Подавать сразу. Детям младшим — порции помельче, можно трубочкой.

## Подготовка заранее

- Не требуется — смузи готовится за минуты. Банан и ягоды можно достать из холодильника/морозилки заранее.

## Состав по компонентам

- **Основа:** банан, ягоды, БЛ йогурт или БЛ молоко.
- **По желанию:** овсяные хлопья, мёд.
- **Подача:** смузи в стакане.

## Порции и время

- На 5 человек. Время: ~5 мин. По 200–250 мл на человека.

## Аллергены и особенности

- БЛ йогурт/молоко. Ягоды, банан. Мёд — детям старше года. Глютен только при добавлении овсянки. ПП.

## Подача

- Холодный смузи. Удобно взять с собой в бутылке. Детям — трубочка.
