# Кабачковые оладьи

Оладьи из кабачка (натереть на тёрке), яйца и муки. Подавать со сметаной БЛ. Хороший способ добавить овощи в завтрак.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Кабачок молодой | 400–500 г (очищенный, без семян) |
| Яйца | 2 шт. |
| Мука пшеничная | 80–120 г |
| Соль | 0,5–1 ч. л. |
| Сода (по желанию, для пышности) | щепотка |
| Масло для жарки | 30–40 г |
| Сметана БЛ для подачи | по вкусу |

## Специи и приправы

- Соль. Перец чёрный — по желанию. Зелень (укроп, петрушка) — в тесто или при подаче. Чеснок в сметану — по желанию.

## Приготовление

1. Кабачок натереть на крупной тёрке (можно с вечера, утром отжать лишний сок). Добавить яйца, соль, соду при желании, перемешать. Всыпать муку — тесто должно быть густым, но не крутым.
2. Разогреть сковороду с маслом. Выкладывать массу столовой ложкой, разравнивать. Жарить на среднем огне 3–4 мин с каждой стороны до румяной корочки.
3. Подавать тёплыми со сметаной БЛ. Детям — небольшие оладушки.

## Подготовка заранее

- **С вечера:** натереть кабачок на тёрке, хранить в миске в холодильнике. Утром отжать лишний сок, добавить яйца, муку и жарить — экономия 5–7 мин.

## Состав по компонентам

- **Основа:** кабачок (тёртый), яйцо, мука.
- **Подача:** оладьи со сметаной БЛ.

## Порции и время

- На 5 человек. Около 25 мин. Кабачок можно натереть с вечера и отжать утром.

## Аллергены и особенности

- Яйцо. Глютен (мука). Сметана — только БЛ. Без молока в тесте.

## Подача

- Тёплые, хрустящие. Детям — небольшие оладушки.
