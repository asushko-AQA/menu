# Гречка по-купечески

Фарш обжаривают с морковью, добавляют гречку и томят под крышкой до готовности. Сытное блюдо в одной сковороде.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Фарш (говядина или свино-говяжий) | 400–500 г |
| Гречка (ядрица) | 300–350 г |
| Морковь | 1–2 шт. (150–200 г) |
| Лук репчатый | 1–2 шт. (150–200 г) |
| Томатная паста | 1–2 ст. л. |
| Вода или бульон | 600–700 мл |
| Масло растительное | 2–3 ст. л. |
| Соль | 1 ч. л. и по вкусу |
| Перец чёрный молотый | по вкусу |
| Лавровый лист | 1 шт. |
| Сметана БЛ при подаче (по желанию) | по вкусу |

## Специи и приправы

- Соль, перец чёрный. Лавровый лист. Чеснок — 1–2 зубчика (по желанию). Сушёная зелень — при подаче. Заготовка: морковь и лук натереть/нарезать с вечера.

## Приготовление

1. Морковь натереть на крупной тёрке, лук нарезать кубиками. В сковороде с высокими бортами или в казанке разогреть масло, обжарить фарш 5–7 мин, разминая. Добавить лук и морковь, обжарить ещё 5 мин. Ввести томатную пасту, перемешать.
2. Гречку промыть, высыпать к мясу. Залить кипящей водой (соотношение с гречкой примерно 1:2), посолить, поперчить, добавить лавровый лист. Довести до кипения, накрыть крышкой и томить на слабом огне 20–25 мин до выпаривания жидкости и готовности гречки.
3. Подавать горячим. По желанию — с ложкой сметаны БЛ. Огурец, зелень — по вкусу.

## Подготовка заранее

- **С вечера:** натереть морковь на тёрке, нарезать лук кубиками; хранить в контейнере в холодильнике. Ужин готовится на 5–7 мин быстрее.

## Состав по компонентам

- **Основа:** фарш (говядина/свинина), гречка, морковь, лук.
- **Подача:** гречка с мясом в тарелках. Огурец, зелень — по желанию.

## Порции и время

- На 5 человек. Около 35–40 мин. Заготовка: морковь и лук с вечера — экономия времени.

## Аллергены и особенности

- Без молочного. Без глютена (гречка). БЛ не требуется.

## Подача

- Горячее. Можно с ложкой сметаны БЛ при подаче.
