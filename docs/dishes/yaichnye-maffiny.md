# Яичные маффины

Смесь для омлета разливают по формочкам для маффинов, добавляют ветчину или сыр (БЛ), запекают. Удобно порционно.

## Ингредиенты (на 5 человек — 10–12 маффинов)

| Продукт | Количество |
|---------|------------|
| Яйца | 8–10 шт. |
| Молоко БЛ | 80–100 мл |
| Ветчина или грудка куриная (нарезка) | 150–200 г |
| Сыр БЛ тёртый | 80–100 г |
| Соль | 0,5 ч. л. |
| Перец чёрный молотый | по вкусу |
| Масло для форм | 10–15 г |

## Специи и приправы

- Соль, перец чёрный. По желанию: сушёная зелень (укроп, петрушка), паприка сладкая.

## Приготовление

1. Формы для маффинов (12 шт.) смазать маслом. Ветчину или курицу нарезать мелкими кубиками. Яйца взбить с молоком, солью и перцем.
2. В каждую форму разложить по 1–2 ст. л. начинки (ветчина/курица), залить яичной смесью почти до краёв, сверху посыпать сыром.
3. Запекать при 180 °C 18–22 мин до застывания и лёгкой румяности. Дать слегка остыть, извлечь из форм. Подавать тёплыми или холодными.

## Состав по компонентам

- **Основа:** яйца, БЛ молоко, ветчина или БЛ сыр.
- **Подача:** маффины из формы.

## Порции и время

- На 5 человек (10–12 маффинов). Около 20–25 мин в духовке.

## Аллергены и особенности

- Яйцо, БЛ молоко/сыр. Без глютена при отсутствии муки. Ветчина — мясо.

## Подача

- Тёплые или холодные. Удобно брать в руку.
