# Макароны по-флотски

Отварные макароны смешивают с обжаренным фаршем и луком. Классическое простое блюдо без соуса. Подают с огурцом или салатом.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Макароны (рожки, перья, спирали) | 400–450 г |
| Фарш (говядина или свино-говяжий) | 500–600 г |
| Лук репчатый | 2 шт. (200–250 г) |
| Масло растительное | 2–3 ст. л. |
| Соль | по вкусу |
| Перец чёрный | по вкусу |
| Зелень для подачи | по вкусу |

## Специи и приправы

- Соль, перец чёрный. Лавровый лист — 1 шт. в фарш при жарке (по желанию). Чеснок — по желанию. Зелень — при подаче.

## Приготовление

1. Макароны отварить по инструкции до аль денте, откинуть на дуршлаг. Лук нарезать кубиками.
2. На сковороде разогреть масло, обжарить фарш 7–10 мин, разминая. Добавить лук, обжарить ещё 5–7 мин до мягкости лука. Посолить, поперчить.
3. Смешать макароны с фаршем в сковороде или в большой миске. Прогреть 1–2 мин. Подавать с огурцом, зеленью или лёгким салатом.

## Подготовка заранее

- **С вечера:** нарезать лук, хранить в холодильнике. Фарш разморозить. Ужин готовится на 5–7 мин быстрее.

## Состав по компонентам

- **Основа:** макароны, фарш, лук.
- **Подача:** макароны по-флотски в тарелке. Огурец, зелень, салат.

## Порции и время

- На 5 человек. Около 25–30 мин.

## Аллергены и особенности

- Глютен (макароны). Без молочного. БЛ не требуется.

## Подача

- Горячие. Классика — с солёным огурцом. Детям — порции без избытка перца.
