# Красная рыба с брокколи

Филе лосося или форели запекают с брокколи в духовке. Минимум ингредиентов, максимум пользы.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Филе лосося или форели (без кожи или со шкуркой) | 500–600 г |
| Брокколи (свежая или замороженная) | 400–500 г |
| Лимон | 0,5–1 шт. |
| Масло растительное или БЛ | 2 ст. л. |
| Соль | по вкусу |
| Перец чёрный | по вкусу |
| Чеснок | 1–2 зубчика |
| Укроп или петрушка (по желанию) | по вкусу |
| Рис или картофель для гарнира | по порциям |

## Специи и приправы

- Соль, перец чёрный. Чеснок, зелень (укроп, петрушка). Лимонный сок при подаче. По желанию: сушёный укроп, паприка сладкая — щепотка на рыбу.

## Приготовление

1. Рыбу нарезать порциями. Брокколи разобрать на соцветия (замороженную не размораживать). Противень застелить пергаментом, сбрызнуть маслом. Выложить рыбу и брокколи. Рыбу сбрызнуть маслом, посолить, поперчить, положить дольки лимона и чеснок (измельчённый или пластинками).
2. Запекать при 180–200 °C 18–25 мин до готовности рыбы (в зависимости от толщины) и мягкости брокколи. Рыба должна легко расслаиваться вилкой.
3. Подавать с рисом или картофелем. Сбрызнуть лимонным соком, посыпать зеленью. Детям — порции без костей.

## Состав по компонентам

- **Основа:** филе красной рыбы, брокколи, лимон, масло, специи.
- **Подача:** рыба + брокколи. Гарнир — рис или картофель.

## Порции и время

- На 5 человек. Около 25–30 мин в духовке. Брокколи можно замороженной.

## Аллергены и особенности

- Рыба. Без глютена. Без молочного в рецепте. БЛ не требуется.

## Подача

- Горячая рыба и брокколи из духовки. Лимон и зелень при подаче.
