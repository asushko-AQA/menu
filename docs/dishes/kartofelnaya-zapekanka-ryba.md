# Картофельная запеканка с рыбой

Слой тёртого картофеля, слой рыбы, заливка из яиц и БЛ молока. Запекают в духовке. Сытная запеканка на всю семью.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Картофель | 1–1,2 кг |
| Филе рыбы (треска, минтай, хек — без костей) | 400–500 г |
| Яйца | 3 шт. |
| Молоко БЛ | 100–150 мл |
| Лук репчатый (по желанию) | 1 шт. (80–100 г) |
| Соль | 0,5–1 ч. л. |
| Перец чёрный | по вкусу |
| Масло БЛ (смазать форму) | 10–15 г |
| Сыр БЛ тёртый (по желанию, сверху) | 50–80 г |

## Специи и приправы

- Соль, перец чёрный. Сушёная зелень (укроп, петрушка) — по вкусу. Паприка сладкая — щепотка в заливку (по желанию). Рыбу можно слегка посолить и поперчить перед укладкой.

## Приготовление

1. Картофель натереть на крупной тёрке (или в комбайне), отжать лишний сок. Рыбу нарезать кусочками, лук — полукольцами (если используете). Форму (примерно 25×30 см) смазать маслом.
2. Выложить половину картофеля, разровнять, посолить. Сверху — слой рыбы (и лука). Накрыть оставшимся картофелем. Яйца взбить с молоком, солью и перцем, залить запеканку равномерно. Сверху посыпать сыром (если используете).
3. Запекать при 180 °C 45–55 мин до румяной корочки и сухой лучинки в центре. Остудить 5–10 мин, нарезать порциями. Подавать с салатом или овощами.

## Состав по компонентам

- **Основа:** картофель (тёртый), филе рыбы, яйца, БЛ молоко. По желанию — лук, БЛ сыр.
- **Подача:** запеканка порциями. Салат или овощи.

## Порции и время

- На 5 человек. Около 45–50 мин (нарезка + запекание). Картофель можно натереть в комбайне. Рыба — без костей.

## Аллергены и особенности

- Только БЛ молоко. Яйцо. Рыба. Без глютена. Подходит при лактозной непереносимости.

## Подача

- Горячая запеканка из духовки. Детям — мягкая текстура, рыба внутри. Остатки хранить в холодильнике.
