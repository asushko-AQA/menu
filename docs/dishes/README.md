# Карточки блюд

Здесь хранятся **карточки отдельных блюд** в формате md.

## Структура карточки

Для каждого блюда создаётся файл с:

- **Название** — краткое, для меню.
- **Краткое описание** — 1–2 предложения для гостя/семьи.
- **Ингредиенты (на 5 человек)** — таблица с продуктами и граммовками (г, мл, шт.). Для супов (обедов) — на **15 порций** (5 чел. × 3 дня).
- **Специи и приправы** — соль, перец, травы, пряности с ориентировочными количествами.
- **Приготовление** — пошаговые действия (2–5 пунктов).
- **Подготовка заранее** — что сделать с вечера или впрок (нарезка, маринад, заморозка), если это возможно; в меню на неделю эти советы переносятся в блок «Что подготовить по дням».
- **Состав по компонентам** — основа, соусы, топпинги, декор.
- **Порции/вес и время** — семья 5 человек, время в минутах.
- **Аллергены и особенности** — лактоза, глютен, БЛ, веган и т.д.
- **Подача** — температура, гарнир, рекомендации для детей.

Имя файла — латиница или транслит, например: `ovsyanaya-kasha.md`, `syrniki-bezlaktoznye.md`.

## Связь с меню

В разделах меню (`docs/menus/*.md`) на блюда даются ссылки вида:

```markdown
[Название блюда](dishes/имя-файла.md)
```

Карточки заполняются по мере разработки меню (субагент «Шеф» или вручную).
