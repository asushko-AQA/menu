# Куриное филе в соево-медовом соусе + рис

Куриное филе нарезают кубиком, быстро обжаривают в соево-медовом соусе. Гарнир — рис.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Куриное филе | 800–1000 г |
| Рис длиннозерный | 300–400 г (сухой вес) |
| Соевый соус | 3–4 ст. л. (45–60 мл) |
| Мёд | 1,5–2 ст. л. (40–50 г) |
| Чеснок | 2–3 зубчика |
| Имбирь свежий (по желанию) | 1 см |
| Масло растительное | 2–3 ст. л. |
| Вода или бульон (для соуса) | 50–80 мл |
| Зелёный лук (при подаче) | по вкусу |

## Специи и приправы

- Чёрный перец молотый, соль — по вкусу. Кунжут (белый/чёрный) — при подаче. Красный перец хлопьями — по желанию (детям не добавлять). Крахмал 1 ч. л. (развести в воде) — по желанию, для загустения соуса.

## Приготовление

1. Рис отварить по инструкции на упаковке (обычно 1:2 с водой, 15–20 мин). Филе нарезать кубиками 2–3 см, замариновать в смеси соевого соуса и мёда (можно с вечера — утром только обжарить).
2. На сковороде разогреть масло, обжарить курицу на сильном огне 5–7 мин до румяности. Добавить измельчённый чеснок и имбирь, перемешать. Влить остаток соуса и воду, потушить 2–3 мин. При желании добавить разведённый крахмал для густоты.
3. Подавать курицу с соусом и рисом. Посыпать кунжутом и зелёным луком. Детям — порции без избытка соуса.

## Подготовка заранее

- **С вечера:** нарезать куриное филе кубиками 2–3 см и замариновать в смеси соевого соуса и мёда. Утром остаётся обжарить и сварить рис.

## Состав по компонентам

- **Основа:** куриное филе (кубик), соевый соус, мёд, чеснок/имбирь.
- **Гарнир:** рис отварной.
- **Подача:** курица в соусе + рис на тарелке.

## Порции и время

- На 5 человек. Около 35–40 мин. Заготовка: филе нарезать и замариновать с вечера.

## Аллергены и особенности

- Соя (соевый соус). Мёд. Без молочного. БЛ не требуется.

## Подача

- Горячее. Рис и курица с соусом. Детям — без избытка соуса.
