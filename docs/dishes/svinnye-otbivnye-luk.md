# Свиные отбивные с луком

Свиную вырезку или корейку нарезают пластами, отбивают, обжаривают с двух сторон, затем тушат с большим количеством лука под крышкой. Мягкие отбивные в луковом соку. Гарнир — картофель, рис или гречка.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Свинина (вырезка, корейка без кости) | 500–600 г |
| Лук репчатый | 3–4 шт. (350–450 г) |
| Мука для панировки | 50–80 г |
| Масло растительное | 2–3 ст. л. |
| Вода или бульон | 50–100 мл |
| Соль, перец | по вкусу |
| Лавровый лист | 1 шт. |
| Картофель, рис или гречка для гарнира | по порциям |
| Зелень для подачи | по вкусу |

## Специи и приправы

- Соль, перец чёрный. Лавровый лист. Чеснок — 1 зубчик (по желанию, в конце). Паприка сладкая — по желанию (посыпать мясо перед жаркой).

## Приготовление

1. Мясо нарезать пластами 1–1,5 см, отбить через плёнку. Посолить, поперчить, обвалять в муке. Лук нарезать полукольцами.
2. Разогреть масло на сковороде. Обжарить отбивные с двух сторон до румяности (2–3 мин с каждой стороны). Выложить. В том же масле обжарить лук 5–7 мин до мягкости.
3. Вернуть отбивные на лук, подлить воду, добавить лавровый лист. Накрыть крышкой и тушить на слабом огне 15–20 мин до мягкости мяса. Подавать с гарниром (пюре, рис, гречка), полить луковым соком. Посыпать зеленью.

## Подготовка заранее

- **С вечера:** нарезать и отбить мясо, нарезать лук; хранить в холодильнике. Ужин готовится на 5 мин быстрее. Гарнир можно сварить заранее.

## Состав по компонентам

- **Основа:** свиные отбивные, лук, мука, бульон/вода.
- **Гарнир:** картофель, рис или гречка.
- **Подача:** отбивная + лук + гарнир. Зелень.

## Порции и время

- На 5 человек. Около 35–40 мин. Заготовка с вечера сокращает время.

## Аллергены и особенности

- Глютен (мука). Без молочного. БЛ не требуется.

## Подача

- Горячие отбивные с луком. Детям — порции без жира по краю.
