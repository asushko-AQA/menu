# Закупки на неделю: продукты и количество

Список продуктов с ориентировочным количеством для семьи из **5 человек** на одну неделю (завтрак + обед + полдник + ужин). Закупки разбиты по неделям в отдельных файлах — удобно подключать к сайту меню (одна страница = одна неделя).

**Важно:** объёмы округлены до удобных для покупки (упаковки, кг). Молочное — только **БЛ** (безлактозное). Ингредиенты из карточек блюд: [docs/dishes/](../dishes/).

### Фрукты на перекусы детям (вне основных приёмов пищи)

Дети едят фрукты просто так — между завтраком, обедом, полдником и ужином. В каждый список по неделям добавлена строка **«Фрукты на перекусы»**: дополнительно **1,5–2 кг** в неделю (яблоки, бананы, груши, мандарины, виноград и т.п.). Держать в доступе в вазе или в холодильнике.

## Как пользоваться

1. **Выбрать неделю** — перейти по ссылке из таблицы выше: [Неделя 1](zakupki-nedelya-1.md), [Неделя 2](zakupki-nedelya-2.md), [Неделя 3](zakupki-nedelya-3.md), [Неделя 4](zakupki-nedelya-4.md).
2. **Фрукты на перекусы** — в каждой неделе заложено **1,5–2 кг** фруктов для перекусов детей вне основных приёмов пищи.
3. **Уточнить по карточкам** — граммовки можно сверить по блюдам в [docs/dishes/](../dishes/) (блок «Ингредиенты»). Супы — на 15 порций (3 дня), печенье/кисель — на 2–3 дня.
4. **Округлить под магазин** — объёмы ориентировочные; округлять до удобных упаковок.
