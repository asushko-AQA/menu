# Жаркое из свинины с кабачками

Свинина кубиками с кабачками и картофелем тушат в одной посуде. Сытное второе без сложного соуса.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Свинина (лопатка, шея — без жира по желанию) | 500–600 г |
| Картофель | 600–800 г |
| Кабачок | 400–500 г |
| Лук репчатый | 1–2 шт. (150–200 г) |
| Морковь | 1 шт. (80–100 г) |
| Томатная паста | 1–2 ст. л. |
| Вода или бульон | 200–300 мл |
| Масло растительное | 2–3 ст. л. |
| Соль | 1 ч. л. и по вкусу |
| Перец чёрный | по вкусу |
| Лавровый лист | 1–2 шт. |
| Чеснок | 2 зубчика |
| Зелень для подачи | по вкусу |

## Специи и приправы

- Соль, перец чёрный. Лавровый лист. Чеснок — в конце. Сушёная зелень (укроп, петрушка) — по желанию. Паприка сладкая — щепотка (по желанию). Заготовка: мясо и овощи нарезать с вечера — ужин готовится быстрее.

## Приготовление

1. Мясо нарезать кубиками 2–3 см. Картофель и кабачок — кубиками. Лук — полукольцами, морковь — кружками или кубиками. В кастрюле с толстым дном или в казане разогреть масло, обжарить мясо до румяности 5–7 мин. Добавить лук и морковь, обжарить 3–4 мин.
2. Добавить томатную пасту, перемешать. Выложить картофель и кабачок, залить водой так, чтобы покрывала примерно половину содержимого. Посолить, поперчить, добавить лавровый лист. Довести до кипения, накрыть крышкой и тушить на слабом огне 35–45 мин до мягкости мяса и овощей. За 5 мин до конца добавить измельчённый чеснок.
3. Подавать горячим. Посыпать зеленью. Детям — порции с мякотью без жира.

## Подготовка заранее

- **С вечера:** нарезать свинину кубиками, картофель и кабачок — кубиками, лук полукольцами, морковь кружками; хранить в контейнерах в холодильнике. Ужин тушится на 10–15 мин быстрее.

## Состав по компонентам

- **Основа:** свинина (кубики), картофель, кабачок, лук, морковь, томатная паста.
- **Подача:** жаркое в тарелках. Зелень, хлеб.

## Порции и время

- На 5 человек. Около 50–60 мин. Заготовка: мясо и овощи нарезать с вечера.

## Аллергены и особенности

- Без молочного. Без глютена. БЛ не требуется.

## Подача

- Горячее жаркое. Зелень при подаче. Детям — без избытка жира.
