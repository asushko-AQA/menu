# Борщ

Классический борщ на мясном бульоне со свёклой, капустой, картофелем и зажаркой. Хранится в холодильнике 2–3 дня, разогревать порциями.

## Ингредиенты (на 15 порций — 5 чел. × 3 дня)

| Продукт | Количество |
|---------|------------|
| Говядина на кости или без (грудинка, лопатка) | 600–800 г |
| Свёкла | 400–500 г |
| Капуста белокочанная | 400–500 г |
| Картофель | 500–600 г |
| Морковь | 200–250 г |
| Лук репчатый | 200–250 г |
| Томатная паста | 2–3 ст. л. |
| Масло растительное | 2–3 ст. л. |
| Сахар | 0,5–1 ч. л. |
| Уксус 9% (по желанию, для цвета свёклы) | 0,5 ч. л. |
| Вода | 3–3,5 л |
| Соль | по вкусу |
| Сметана БЛ при подаче | по порциям |
| Зелень (укроп, петрушка) | по вкусу |

## Специи и приправы

- Соль, перец чёрный горошком — 3–5 шт. Лавровый лист — 2 шт. Чеснок — 2–3 зубчика (в конце или при подаче). Зелень — при подаче.

## Приготовление

1. Мясо залить холодной водой, довести до кипения, снять пену. Варить 1–1,5 ч на слабом огне под крышкой. Бульон посолить. Мясо вынуть, нарезать кусочками, вернуть в бульон (или положить в суп при разогреве).
2. Картофель нарезать кубиками, капусту нашинковать. Добавить в кипящий бульон: сначала картофель, через 5–7 мин — капусту. Варить до мягкости.
3. Зажарка: лук и морковь нарезать, свёклу натереть. Обжарить на масле 5–7 мин, добавить томатную пасту, сахар, чуть уксуса, потушить 3–4 мин. Выложить зажарку в суп, добавить лавровый лист и перец. Варить 5–10 мин. Попробовать, посолить. Чеснок и зелень — при подаче.
4. Хранить в холодильнике в кастрюле с крышкой 2–3 дня. Разогревать нужное количество в кастрюле или порциями в микроволновке. Подавать со сметаной БЛ и зеленью.

## Подготовка заранее

- **В день варки (воскресенье или среда):** с вечера предыдущего дня можно нарезать овощи (картофель хранить в воде), натереть свёклу и морковь — хранить в контейнерах в холодильнике. Утром сварить бульон и собрать борщ.

## Состав по компонентам

- **Бульон:** мясо, вода, соль.
- **Овощи:** свёкла, капуста, картофель, морковь, лук.
- **Зажарка:** морковь, лук, свёкла, томатная паста.
- **Подача:** сметана БЛ, зелень, хлеб.

## Порции и время

- На 15 порций (5 человек × 3 дня). Время: ~50–60 мин (с учётом варки бульона 1–1,5 ч можно заранее сварить мясо). Хранение в холодильнике 2–3 дня.

## Аллергены и особенности

- Без глютена. Сметана при подаче — только БЛ. Сельдерей не используется.

## Подача

- Горячий борщ. Сметана БЛ и зелень в тарелку. Хлеб. Детям — порции без избытка перца.
