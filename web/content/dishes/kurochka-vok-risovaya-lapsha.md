# Курица с овощами и рисовой лапшой (вок)

Быстрое блюдо в азиатском стиле: куриное филе, замороженная овощная смесь или свежие перец и морковь, соевый соус и **рисовая лапша**. Без остроты — подходит детям.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Куриное филе | 700–800 г |
| Рисовая лапша (сухая) | 200–250 г |
| Овощи (перец, морковь, брокколи, стручковая фасоль) или смесь замороженная | 400–600 г |
| Чеснок | 2 зубчика |
| Имбирь свежий (по желанию) | 1 см |
| Соевый соус | 3–4 ст. л. (45–60 мл) |
| Масло растительное | 2–3 ст. л. |
| Вода или бульон | 50–100 мл |
| Кунжут (при подаче) | по желанию |
| Зелёный лук | по желанию |

## Специи и приправы

- По желанию крахмал 1 ч. л., разведённый в воде, для густого соуса. Красный перец не добавлять детям.

## Приготовление

1. Лапшу отварить по упаковке на **1–2 мин** меньше аль денте, промыть холодной водой. Филе нарезать полосками. Овощи нарезать соломкой (если свежие).
2. Вок или большую сковороду разогреть с маслом. Обжарить курицу на сильном огне **5–6 мин**. Добавить чеснок и имбирь, затем овощи — **4–5 мин** до мягкости.
3. Влить соевый соус и немного воды, перемешать. Добавить лапшу, прогреть **1–2 мин**. Посыпать кунжутом и луком.

## Подготовка заранее

- Нарезать курицу и овощи с вечера.

## Состав по компонентам

- **Основа:** курица, овощи, рисовая лапша.
- **Соус:** соевый соус.

## Порции и время

- На 5 человек. ~25–30 мин.

## Аллергены и особенности

- Соя. Без молочного. Лапша — проверьте состав на глютен (рисовая обычно без глютена).

## Подача

- Горячее. Детям — порции без избытка соуса.
