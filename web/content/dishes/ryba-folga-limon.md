# Рыба в фольге с лимоном

Филе белой или красной рыбы запекают в фольге с лимоном и травами. Нежно, сочно, без лишнего жира.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Филе рыбы (треска, минтай, хек, форель) | 500–600 г |
| Лимон | 1 шт. |
| Масло растительное или БЛ | 2 ст. л. |
| Соль | по вкусу |
| Перец чёрный | по вкусу |
| Чеснок | 2 зубчика |
| Укроп или петрушка (свежие или сушёные) | по вкусу |
| Прованские травы (по желанию) | 0,5 ч. л. |
| Овощи для гарнира (картофель, рис) | по порциям |

## Специи и приправы

- Соль, перец чёрный. Чеснок, укроп/петрушка. Прованские травы, лимон — по желанию. Можно добавить тонкие дольки лимона и веточку укропа в каждый конверт.

## Приготовление

1. Рыбу нарезать порциями. Каждый кусок сбрызнуть маслом, посолить, поперчить, положить сверху дольку лимона, измельчённый чеснок и зелень. Завернуть в фольгу (конвертиком или кармашком), чтобы сок не вытекал.
2. Противень застелить пергаментом. Выложить конверты. Запекать при 200 °C 20–25 мин (в зависимости от толщины рыбы). Рыба готова, когда легко расслаивается вилкой.
3. Подавать в фольге или аккуратно открыть и переложить на тарелку с гарниром (картофель, рис, овощи). Полить соком из фольги. Детям — проверить на кости.

## Состав по компонентам

- **Основа:** филе рыбы, лимон, масло, чеснок, зелень.
- **Подача:** рыба в фольге с соком. Гарнир — картофель или рис.

## Порции и время

- На 5 человек. Около 25–30 мин. Удобно: каждый порционный конверт — своя порция.

## Аллергены и особенности

- Рыба. Без глютена. Без молочного. БЛ не требуется.

## Подача

- Горячая рыба из фольги. Сок не выливать — полить сверху. Гарнир по желанию.
