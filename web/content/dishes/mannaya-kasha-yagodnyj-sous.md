# Манная каша с ягодным соусом

Классическая манная каша на БЛ молоке, подаётся с соусом из размороженных или свежих ягод (малина, черника, клубника). Без комков — варить, помешивая.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Манка | 120–150 г |
| Молоко БЛ | 700–800 мл |
| Сахар | 2–3 ст. л. (по вкусу) |
| Соль | щепотка |
| Масло БЛ | 20–30 г |
| Ягоды замороженные или свежие | 250–350 г |
| Сахар для соуса | 1 ст. л. (по вкусу) |
| Крахмал (для загустения соуса, по желанию) | 0,5 ч. л. |

## Специи и приправы

- Соль. Ванилин — по желанию (в кашу). В соус — только сахар по вкусу.

## Приготовление

1. Молоко довести до кипения, посолить, добавить сахар. Тонкой струйкой всыпать манку, постоянно помешивая. Варить на слабом огне 5–7 мин до загустения. Добавить масло, перемешать.
2. Ягоды разморозить (или использовать свежие), при желании пюрировать блендером. Подсластить по вкусу. Если нужен густой соус — прогреть с разведённым крахмалом 1–2 мин.
3. Разложить кашу по тарелкам, сверху полить ягодным соусом. Подавать тёплой.

## Подготовка заранее

- **Впрок:** ягодный соус можно сварить с вечера и хранить в холодильнике. Утром разогреть кашу или сварить свежую.

## Состав по компонентам

- **Основа:** манка, БЛ молоко, сахар, масло.
- **Топпинг:** ягодный соус (малина, черника, клубника).
- **Подача:** каша в тарелке с соусом сверху.

## Порции и время

- На 5 человек. Около 15–20 мин.

## Аллергены и особенности

- БЛ молоко. Глютен (манка). Ягоды — возможная аллергия у детей.

## Подача

- Тёплая каша с ягодным соусом. Детям — порции помельче.
