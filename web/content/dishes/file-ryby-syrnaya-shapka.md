# Филе рыбы под сырной шубкой

Филе белой рыбы запекают под «шубкой» из тёртого БЛ сыра с майонезом или сметаной БЛ и чесноком. Семейный вариант.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Филе белой рыбы (треска, минтай, хек) | 500–600 г |
| Сыр БЛ тёртый | 150–200 г |
| Сметана БЛ или майонез | 3–4 ст. л. (80–100 г) |
| Чеснок | 2–3 зубчика |
| Соль, перец | по вкусу |
| Укроп или петрушка (по желанию) | по вкусу |
| Лимон (сбрызнуть рыбу) | 2–3 дольки |
| Масло для противня | 1 ст. л. |
| Рис или картофель для гарнира | по порциям |

## Специи и приправы

- Соль, перец чёрный. Чеснок в «шубку». Сушёная зелень — по желанию. Рыбу слегка посолить и сбрызнуть лимоном перед нанесением шубки.

## Приготовление

1. Рыбу нарезать порциями, посолить, поперчить, сбрызнуть лимонным соком. Выложить на смазанный маслом противень. Духовку разогреть до 180 °C.
2. Смешать тёртый сыр БЛ со сметаной (или майонезом), измельчённым чесноком и зеленью. Равномерно выложить «шубку» на каждую порцию рыбы.
3. Запекать 25–35 мин до готовности рыбы (она должна расслаиваться вилкой) и румяной сырной корочки. Подавать с рисом или картофелем. Детям — порции без избытка чеснока, проверить на кости.

## Подготовка заранее

- **С вечера:** натереть сыр на тёрке, хранить в контейнере в холодильнике. Смесь для «шубки» (сыр + сметана + чеснок) можно приготовить с вечера и нанести на рыбу перед запеканием.

## Состав по компонентам

- **Основа:** филе рыбы, БЛ сыр, сметана БЛ (или майонез), чеснок.
- **Подача:** рыба под сырной шубкой. Гарнир — рис или картофель.

## Порции и время

- На 5 человек. Около 35–40 мин. Сыр можно натереть с вечера.

## Аллергены и особенности

- Рыба. Только БЛ сыр и БЛ сметана. Без глютена в блюде. Яйцо в майонезе — при использовании.

## Подача

- Горячая рыба из духовки с хрустящей сырной корочкой. Гарнир по желанию.
