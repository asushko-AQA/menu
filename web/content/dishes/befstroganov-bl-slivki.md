# Бефстроганов на безлактозных сливках + пюре

Тонко нарезанное мясо тушат с луком и БЛ сливками. Подают с пюре на БЛ молоке. Классика с учётом диеты.

## Ингредиенты (на 5 человек)

| Продукт | Количество |
|---------|------------|
| Говядина (вырезка, бедро без кости) | 500–600 г |
| Лук репчатый | 2 шт. (200–250 г) |
| Сливки БЛ 10–20% | 200–250 мл |
| Томатная паста | 1–2 ст. л. |
| Мука (для пассеровки, по желанию) | 1 ст. л. |
| Масло БЛ или растительное | 2–3 ст. л. |
| Соль, перец | по вкусу |
| Картофель для пюре | 1–1,2 кг |
| Молоко БЛ для пюре | 100–150 мл |
| Масло БЛ для пюре | 30–40 г |

## Специи и приправы

- Соль, перец чёрный. Мускатный орех — щепотка в сливки (по желанию). В пюре: соль. Мясо нарезать тонко — быстрее готовится и нежнее.

## Приготовление

1. Мясо нарезать тонкими полосками поперёк волокон. Лук — полукольцами. Картофель отварить до мягкости, приготовить пюре на БЛ молоке и масле, посолить.
2. На сковороде разогреть масло, обжарить мясо партиями на сильном огне 2–3 мин до румяности, выложить. В том же масле обжарить лук до мягкости. Вернуть мясо, добавить томатную пасту, перемешать. Влить сливки БЛ, посолить, поперчить. Тушить на слабом огне 5–8 мин. При желании загустить мукой (развести в холодной воде, влить и прогреть).
3. Подавать бефстроганов с пюре. Детям — порции без избытка перца.

## Подготовка заранее

- **С вечера:** нарезать мясо тонкими полосками поперёк волокон, хранить в холодильнике. Лук нарезать полукольцами. Ужин собирается на 5–7 мин быстрее. Пюре можно сварить заранее и разогреть.

## Состав по компонентам

- **Основа:** говядина (тонкая нарезка), лук, БЛ сливки 10–20%, томатная паста.
- **Гарнир:** пюре (картофель, БЛ молоко, БЛ масло).
- **Подача:** бефстроганов + пюре на тарелке.

## Порции и время

- На 5 человек. Около 35–40 мин. Мясо нарезать тонко — быстрее готовится.

## Аллергены и особенности

- Только БЛ сливки, БЛ молоко и масло в пюре. Глютен не обязателен (мука по желанию).

## Подача

- Горячее. Пюре нежное, соус сливочный. Детям — без избытка перца.
