# Чечевичный суп с овощами

Сытный суп из красной или зелёной чечевицы с морковью, луком и картофелем. Можно без мяса или с небольшим количеством копчёностей для аромата. На **15 порций** (5 человек × 3 дня).

## Ингредиенты (на 15 порций — 5 чел. × 3 дня)

| Продукт | Количество |
|---------|------------|
| Чечевица красная (быстроразваривающаяся) или зелёная | 350–450 г |
| Картофель | 500–600 г |
| Морковь | 250–300 г |
| Лук репчатый | 250–300 г |
| Чеснок | 2–3 зубчика |
| Томатная паста | 1–2 ст. л. |
| Масло растительное | 2 ст. л. |
| Вода или лёгкий бульон | 3–3,5 л |
| Копчёности (грудинка, рёбрышко) — по желанию | 100–150 г |
| Соль, перец | по вкусу |
| Зелень (укроп, петрушка) | по вкусу |

## Специи и приправы

- Соль, перец чёрный горошком 3–4 шт., лавровый лист 1–2 шт. Кумин — щепотка по желанию. Зелень при подаче.

## Приготовление

1. Чечевицу промыть. Картофель нарезать кубиком. В кастрюле разогреть масло, обжарить лук и морковь 5–7 мин, добавить томатную пасту и чеснок, перемешать.
2. Залить водой или бульоном, довести до кипения. Если используете копчёности — добавить целиком для аромата или мелко нарезать и положить вместе с картофелем.
3. Добавить картофель и чечевицу, уменьшить огонь. Варить **15–25 мин** (красная чечевица быстрее) до мягкости картофеля и разваривания чечевицы. Посолить, поперчить, лавровый лист за 5 мин до конца вынуть.
4. Хранить в холодильнике до **2–3 дней**, разогревать порциями. При загущении разбавить кипятком.

## Подготовка заранее

- С вечера нарезать морковь и лук, картофель — в контейнер с водой. Утром собрать суп за ~30 мин.

## Состав по компонентам

- **Основа:** чечевица, картофель, зажарка из моркови и лука.
- **По желанию:** копчёности.
- **Подача:** зелень, хлеб.

## Порции и время

- На 15 порций. Время: ~35–45 мин.

## Аллергены и особенности

- Без глютена. Без молочного. Бобовые (чечевица). Копчёности — по желанию.

## Подача

- Горячий суп. Детям — порции без избытка перца.
